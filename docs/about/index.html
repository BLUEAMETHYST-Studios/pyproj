Pyproj

About

Pyproj is a tool to build Python applications and packages developed by BLUEAMETHYST Studios.
The tool was programmed in Python and is being built with PyInstaller.
Pyproj started as a small project at BLUEAMETHYST in the Mid-Summer of 2023 (1st of July 2023).
The Pyproj project is non-profit and open-source as it was developed by BLUEAMETHYST Studios,
which project are free and open-source.
